How was Pharma AI made?

Pharma AI was made using VS CODE. The Programming language that was included within the VS CODE to make the website was HTML, CSS, and Javascript. I have many experience using VS CODE and this project is not the only project I have made. My first step in making the project was planning out how the design of the website would look like. I would then first start off making the HTML part which is the structure of the website. I would then add designs by CSS and alter the look of the website. One example is the background images/color of texts. I then added an animation made possible by Javascript an example is the typing text you can see at the homepage. For the AI to work Javascript was also needed. The Javascript code stored the possible questions a user can ask and then had answers for these questions. This code was connected to a container in which the person can interact with known as the chatbox made in the CSS section. If a person were to ask a question the Javascript code would find a common asked question thats similair to the one the user asked and then generate the answer to that question.
